Kitten-Flask es un mini CRUD hecho en Flask con un enfoque en seguridad, clean architecture y buenas prácticas para evitar concurrencia, SQL Injection y acoplamiento fuerte.

Descripcion

Este proyecto ejemplifica la construcción de una API básica para administrar recursos de “gatos” (cats) con las siguientes características:

  • - Arquitectura en capas (presentación, servicio, repositorio, base de datos)
  • - Uso de Pydantic para validación tipada y modelos (entrada/salida)
  • - Manejo seguro de la conexión a base de datos por petición (thread-safe)
  • - SQL parametrizado para evitar inyecciones
  • - Manejo de errores / excepciones customizadas
  • - Separación de responsabilidades y bajo acoplamiento
